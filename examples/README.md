# サンプルファイル

このディレクトリにはテスト用のサンプル音声ファイルとテキストを配置できます。

## 使用例

1. 音声ファイル（例：`sample.wav`）を用意
2. 対応するテキストファイル（例：`sample.txt`）を作成
3. アプリケーションで音声ファイルをアップロードし、テキストを入力
4. 生成されたTextGridファイルをPraatで確認

## サンプルテキスト

```
こんにちは。今日は良い天気ですね。
```

```
音声認識技術の発展により、様々な応用が可能になりました。
```

## 注意事項

- 音声ファイルは著作権に注意してください
- 個人情報を含む音声ファイルは使用しないでください
- サンプルファイルはGitリポジトリにコミットしないでください（.gitignoreで除外）
